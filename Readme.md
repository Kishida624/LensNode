## Running Application


```
npm install
npm start
```



Hit application at 
```
`http://localhost:5000`
```

Imprtant: Application is using nodemon, to continuously running on port 5000, unless terminated (Ctrl + C). This is to help identify dev time error and not getting into issue of running it from scratch again.
